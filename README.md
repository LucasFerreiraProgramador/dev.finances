## 💻 Projeto
O **dev.finances** é uma aplicação de controle financeiro, onde é possível cadastrar e excluir transações,
além de visualizar o saldo de entradas e saídas.

Projeto de prática pessoal, com foco em lógica de programação,
manipulação do DOM e organização de dados no front-end.
